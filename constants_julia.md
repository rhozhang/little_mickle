# built-in constants in julia

tag: tool

> Many a little makes a mickle

- circumference ratio (3.1415...):     pi or  π (REPL: \pi<tab>)
- ℯ (2.71828...):    ℯ (REPL: \euler<tab>), or Base.MathConstants.e
- imaginary unit:   im
- golden ratio: Base.MathConstants.golden

```jl
im^2   # -1 + 0im
```
